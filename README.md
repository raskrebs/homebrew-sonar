# homebrew-sonar

Homebrew tap for [sonar](https://github.com/raskrebs/sonar) — a CLI for monitoring local ports and processes.

## Install

```sh
brew install raskrebs/sonar/sonar
```

Or tap first, then install:

```sh
brew tap raskrebs/sonar
brew install sonar
```

## Upgrade

```sh
brew update
brew upgrade sonar
```

## Uninstall

```sh
brew uninstall sonar
brew untap raskrebs/sonar
```

## Supported platforms

- macOS (arm64, amd64) — includes `sonar-tray`
- Linux (arm64, amd64)
